### 演變過程

**第一階段：有無旋鈕？**

- Joe 提議：兩個掣（Button A = AI Mode，Button B = Bluetooth Mode）

**第二階段：點解要旋鈕？**

- 浴室環境需要音量控制（花灑聲覆蓋 / 深夜要靜）
- Joe 確認：「**照用旋轉按鈕**」

**第三階段：形態確認**

- Joe：「**只有一個圓形，其他乜都唔要**」
- 確認：The One Knob

### 最終硬件形態

| 項目 | 規格 |
|------|------|
| **面板** | 86 標準入牆面板 |
| **核心** | 一個懸浮光環旋鈕（中央） |
| **形態** | 零屏幕、零按鍵、零文字 |
| **物料** | 黑鏡浮光（黑色鋼化玻璃 + 鋁合金旋鈕） |

### 操作方式

| 動作 | 功能 |
|------|------|
| **轉動** | 音量控制 |
| **短按** | 喚醒 AI / 播放暫停 |
| **長按（3秒）** | 切換模式（AI ↔ 藍牙） |

## Session G：藍牙模式確認

**時間**：約 22:00

### 藍牙特點（Joe 堅持）

- **自家研發藍牙 Stack**（非 generic）
- **自訂名稱**：跟 [座數-單位] 命名（如 Tower1-8A Soulmate）
- **防誤連**：需實體旋鈕授權配對

### 啞巴模式（Mute Mode）

- 預設狀態：絕對靜音（無任何 system beep）
- 只有用戶主動操作才發聲
- 防止半夜突然播放嚇親人
